# 起動

## 機能

## 起動シーン

- 起動画面をアニメ表示
- 起動画面の表示秒数の設定
- 起動画面の表示中に、操作でスキップ
- 保存状態の読み込みと初期化
- 選択したシーンを読み込んで切り替え

### タイトルシーン

- シーン切り替えのものを利用する
- ユーザー設定の状態を、タイトルとして保存
- ボタンで、ゲーム開始

### ゲームシーン

- ユーザー設定の状態を、ゲームとして保存
- 倒したオブジェクトデータがあったら、該当するオブジェクトを削除
- 倒したオブジェクトのシーンパスを保存
- ボタンで、タイトルに戻る
- ボタンで、リトライして初期化。倒したオブジェクトをリセット
